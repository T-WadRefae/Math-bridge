# جسر الرياضيات 🌈 — جزيرة المغامرات الحسابية

منصة تعليمية مرحة لتعويض الفاقد في الرياضيات لطلبة الصفوف ٥–٩ في فلسطين.
إعداد وإشراف: المعلمة **T. Wad Refae**

## الصفحات
- `index.html` — الصفحة الرئيسية (الجزيرة، العوالم، مهام الإنقاذ، رحلتي…)
- `diagnose.html` — **اختبار البداية التشخيصي** (٦ أسئلة عبر عالمي الأعداد ثم الكسور)
- `rescue-numbers.html` — **مهمة إنقاذ الأعداد** (علاج القيمة المنزلية بالبطاقات)
- `rescue.html` — **مهمة إنقاذ الكسور** (علاج جمع الكسور بالأشرطة)
- `app.js` — البطاقات + التفاعلات + لوحة المعلّم المخفية

## الرحلة التعليمية الكاملة
```
التشخيص (٦ أسئلة)  →  تحليل النتائج  →  التوجيه للعلاج الأنسب  →  شهادة الإتقان
   diagnose.html        (نسبة كل عالم)     (الأضعف تلقائياً)         (فتح المستوى التالي)
```

### التشخيص الذكي — عالمان، ٦ مهارات
| العالم | المهارة | السؤال | الخطأ يكشف فجوة |
|--------|---------|--------|------------------|
| 🔢 الأعداد | القيمة المنزلية | قيمة ٣ في ٤٣٧ | خلط المنازل |
| 🔢 الأعداد | ترتيب العمليات | ٢+٣×٤ | الحساب يساراً ليمين |
| 🔢 الأعداد | الطرح بالاستلاف | ٥٠٢−٢٧٨ | خطأ الاستلاف عبر الصفر |
| 🍰 الكسور | جمع المتشابهة | ½+½ | جمع المقامات |
| 🍰 الكسور | مقارنة الكسور | ⅓ أم ¼ | المقام الأكبر = أكبر |
| 🍰 الكسور | كسر من كمية | ربع الـ٢٠ | القسمة الخاطئة |

**التوجيه:** بعد الأسئلة الستة، تُحسب نسبة كل عالم، ويُوجَّه الطالب تلقائياً
لعلاج **العالم الأضعف** (والمهارة الأكثر تعثّراً فيه):
- الأعداد أضعف → `rescue-numbers.html`
- الكسور أضعف → `rescue.html`

### مهام الإنقاذ — مسار من ٦ خطوات
ترحيب → شرح بصري → تدريب موجّه → تدريب ذاتي → **تحدٍّ بنفس سؤال البداية** → شهادة بـ٣ نجوم.
الأخطاء أثناء التدريب تُظهر تلميحاً وتسمح بإعادة المحاولة (تعلّم بلا عقاب).

## النشر على GitHub Pages
ارفعي كل الملفات للمستودع، ثم Settings → Pages → الفرع `main`. تعمل مباشرة (Tailwind عبر CDN).

## لوحة المعلّم المخفية 🔐
رابط خاص فقط: `.../index.html#/teacher/wadrefae-2026`
لتغيير المفتاح: عدّلي `const TEACHER_KEY` في `app.js`.

> ملاحظة: الإخفاء عبر الرابط ليس حماية أمنية. لحماية بيانات الطلبة فعلياً يلزم تسجيل دخول حقيقي عبر خادم لاحقاً.
