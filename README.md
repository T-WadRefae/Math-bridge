# جسر الرياضيات 🌈 — جزيرة المغامرات الحسابية

منصة تعليمية مرحة لتعويض الفاقد في الرياضيات لطلبة الصفوف ٥–٩ في فلسطين.
إعداد وإشراف: المعلمة **T. Wad Refae**

## الصفحات
- `index.html` — الصفحة الرئيسية (الجزيرة، العوالم، مهام الإنقاذ، رحلتي…)
- `diagnose.html` — **اختبار البداية التشخيصي** (كل خيار يكشف فجوة)
- `rescue.html` — **مهمة إنقاذ الكسور** (رحلة علاج تفاعلية بالصور)
- `app.js` — البطاقات + التفاعلات + لوحة المعلّم المخفية

## الرحلة التعليمية الكاملة (نموذج العرض)
```
التشخيص (½+½)  →  اكتشاف الفجوة  →  مهمة الإنقاذ  →  شهادة الإتقان
   diagnose.html      (تقرير)          rescue.html       (فتح المستوى التالي)
```

### ١) التشخيص الذكي — منطق الـ distractors
السؤال: **½ + ½ = ؟** — كل خيار خاطئ يكشف فجوة ويوجّه لها:

| الخيار | نوع الخطأ | الفجوة المكتشفة |
|--------|-----------|------------------|
| **٢/٤** | جمع البسط والمقام معاً | الخلط في طريقة جمع الكسور |
| **¼**  | ضرب المقامات بدل الجمع | الخلط بين الجمع والضرب |
| **٢/٢** | لم يبسّط الكسر | عدم إدراك أن ٢/٢ = ١ |
| **١** ✓ | الإجابة الصحيحة | الإتقان |

عند الخطأ يظهر زر «ابدأ مهمة الإنقاذ» الذي ينقل الطالب إلى `rescue.html`.

### ٢) مهمة الإنقاذ — العلاج بالصور
مسار من ٦ خطوات: ترحيب → شرح بصري بالبيتزا/الأشرطة → تدريب موجّه →
تدريب ذاتي → **التحدّي النهائي (نفس سؤال البداية ½+½)** → شهادة إتقان بـ٣ نجوم.
الأخطاء أثناء التدريب تُظهر تلميحاً وتسمح بإعادة المحاولة (تعلّم بلا عقاب).

## النشر على GitHub Pages
ارفعي كل الملفات للمستودع، ثم Settings → Pages → الفرع `main`. تعمل مباشرة (Tailwind عبر CDN).

## لوحة المعلّم المخفية 🔐
رابط خاص فقط (غير مرتبط بأي زر ظاهر):
```
.../index.html#/teacher/wadrefae-2026
```
لتغيير المفتاح: عدّلي `const TEACHER_KEY` في `app.js`.

> ملاحظة: الإخفاء عبر الرابط ليس حماية أمنية. لحماية بيانات الطلبة فعلياً يلزم تسجيل دخول حقيقي عبر خادم لاحقاً.
