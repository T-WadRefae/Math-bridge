# جسر الرياضيات 🌈 — جزيرة المغامرات الحسابية

منصة تعليمية مرحة لتعويض الفاقد في الرياضيات لطلبة الصفوف ٥–٩ في فلسطين.
إعداد وإشراف: المعلمة **T. Wad Refae**

## المنهجية المرجعية 📚
- [`docs/methodology.md`](docs/methodology.md) — **منهجية فحص ومتابعة وحل الفاقد التعليمي** (صفوف ٥–٩):
  دورة الفاقد الرباعية، خريطة المنهاج الفلسطيني، سلاسل المهارات التأسيسية الحرجة،
  بنك التشخيص، مسارات العلاج، أدوات المعلّمة، وخارطة ربطها بالمنصة.

## الصفحات والبنية
- `curriculum.js` — **المنهاج المركزي المدفوع بالبيانات**: ٦ عوالم × صفوف ٥–٩ (٤٤ مهارة تأسيسية)،
  لكل مهارة سؤال تشخيصي بمشتّتات كاشفة للفجوة + محتوى علاجي (شرح + تدريبان). التوسّع = إضافة كائن مهارة فقط.
- `index.html` — الصفحة الرئيسية (الجزيرة، العوالم، مهام الإنقاذ، رحلتي…)
- `diagnose.html` — **اختبار البداية التشخيصي حسب الصف**: يختار الطالب صفّه فيُولَّد اختبار متوازن عبر العوالم.
- `rescue-skill.html?id=<المهارة>` — **محرّك العلاج العام**: يبني مسار الخطوات الستّة لأي مهارة من بياناتها.
- `rescue-numbers.html` — تجربة بصرية مصمّمة يدوياً (القيمة المنزلية بالبطاقات).
- `rescue.html` — تجربة بصرية مصمّمة يدوياً (جمع الكسور بالأشرطة).
- `app.js` — بطاقات الرئيسية + التفاعلات + لوحة المعلّم المخفية.

## الرحلة التعليمية الكاملة
```
اختيار الصف  →  تشخيص متوازن  →  تحليل + العالم الأضعف  →  علاج المهارة  →  إتقان بـ٣ نجوم  →  المهارة التالية
 diagnose.html   (حسب المنهاج)      (تلقائياً)              rescue-skill.html   (يُحفظ محلياً)     (سلسلة التتابع)
```

### التشخيص الذكي — مدفوع بالمنهاج
- يختار الطالب صفّه (٥–٩)، فتُسحب مهارات ذلك الصف من `curriculum.js` ويُبنى اختبار **متوازن يوزّع الأسئلة على العوالم**.
- كل خيار خاطئ موسوم بـ**اسم الفجوة** التي يكشفها، وتُعرض تغذية راجعة فورية تشرح سبب الخطأ.
- تُحسب نسبة كل عالم، ويُوجَّه الطالب تلقائياً لعلاج **العالم الأضعف** بدءاً بأول **مهارة متعثّرة** فيه.
- النتائج تُحفظ في `localStorage` (المفتاح `mb_state`) تمهيداً لتفعيل «رحلتي» ولوحة المعلّم ببيانات حقيقية.

### مهام الإنقاذ — مسار من ٦ خطوات (`STEPS`)
`intro` ترحيب → `learn` شرح المفهوم → `practice1` تدريب موجّه → `practice2` تدريب ذاتي →
`challenge` **تحدٍّ بنفس سؤال التشخيص** (قياس بعدي) → `done` شهادة بـ٣ نجوم + تسجيل الإتقان + المهارة التالية.
الأخطاء أثناء التدريب تُظهر تلميحاً وتسمح بإعادة المحاولة (تعلّم بلا عقاب).

## النشر على GitHub Pages
ارفعي كل الملفات للمستودع، ثم Settings → Pages → الفرع `main`. تعمل مباشرة (Tailwind عبر CDN).

## لوحة المعلّم المخفية 🔐
رابط خاص فقط: `.../index.html#/teacher/wadrefae-2026`
لتغيير المفتاح: عدّلي `const TEACHER_KEY` في `app.js`.

> ملاحظة: الإخفاء عبر الرابط ليس حماية أمنية. لحماية بيانات الطلبة فعلياً يلزم تسجيل دخول حقيقي عبر خادم لاحقاً.
